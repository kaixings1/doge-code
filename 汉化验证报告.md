### 1. 源码性质
所有 `src` 目录下的文件都是**编译后的产物**（包含 React Compiler 的 `_c` 缓存机制和 base64 source maps），而非原始 TypeScript/TSX 源文件。这意味着：
- 直接编辑这些文件会在下次编译时被覆盖
- 需要找到原始源文件进行翻译，然后重新编译

### 4. 不应翻译的内容

以下内容**保持英文**是正确的：
- 环境变量名（如 `DOGE_API_KEY`, `ANTHROPIC_API_KEY`）
- 代码标识符、函数名、变量名
- API 端点 URL
- GitHub Issue 模板内容（为国际化考虑）
- 系统 prompt（用于 AI 模型，非用户可见）
- 日志标签（如 `logLabel: 'output-styles'`）
- 技术协议/格式名称（如 `OAuth`, `API`, `MCP`, `Bedrock`, `Vertex`）
