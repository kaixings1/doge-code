游戏已启动，基本功能已实现。

规则:
1. 飞机滚动，敌机滚动
2. 碰撞打印 "碰撞！" 并结束游戏
3. 飞机超出边界打印 "飞机超出边界！游戏结束"
4. 分数计数每次循环增加 1

如何应用:
- 当用户说 "继续游戏" 时，重新开始游戏
- 当用户说 "查看记录" 时，显示游戏信息
- 当用户说 "结束游戏" 时，清除记录并退出

为什么:
- 这是一个简单的命令行游戏，需要记忆游戏进度
- 玩家可能想重玩或查看结果
- 记录有助于理解游戏逻辑

如何应用:
- 使用 Read 读取 memory.md
- 使用 Write 写入新游戏数据
- 使用 Edit 修改记录
