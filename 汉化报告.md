## 三、汉化策略建议

### 1. 使用翻译键
```typescript
// ❌ 避免硬编码
export const HELLO_WORLD = "Hello, World!";

// ✅ 使用翻译键
export const T_HELLO_WORLD = "T_HELLO_WORLD"; // 翻译键
export const t = () => "你好，世界！"; // 翻译函数
```

### 2. 保持对照
在关键文件顶部添加中英文对照表：
```typescript
/**
 * 汉化对照表:
 * T_HELLO_WORLD: "Hello, World!" -> "你好，世界！"
 * T_CMD_HELP: "Show help" -> "显示帮助"
 */
```

### 3. 优先汉化顺序
1. 用户可见的 UI 文本
2. 命令帮助信息
3. 错误提示信息
4. 状态消息
5. 注释和文档
